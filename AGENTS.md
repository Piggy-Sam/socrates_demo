# Socrates AI — design language ("The Instrument") — CURRENT, authoritative

This is the live design system (shipped to prod from `main` → socrates-demo-chi.vercel.app). It SUPERSEDES the original SPEC.md §9 "Aegean Night / gold / Fraunces-Spectral / constellation" direction, which is retired. SPEC.md §2 (the Socratic identity / system prompt) is the unchanging soul — never alter it.

**Aesthetic:** a scientific instrument / refined terminal for thinking. **Light-first** (default), high contrast, hairline rules, `FIG.`-style mono captions, command-line cues. Modern, minimal; terminal cues are the delight in the details.

**Color (tokens in `app/globals.css`; legacy names kept and ALIASED to avoid churn — `--gold`/`--cyan` → `--accent`):** one profound blue `--accent` `#0F62FE` on near-white paper `--ink` `#F4F6FA`, near-black text `--marble` `#0B0F1A`, hairline `#D7DEE8`. **Dark** = deep ink-navy invert (`--ink` `#0B1220`, `--accent` `#4D7CFF`, `--marble` `#E8EDF5`). Accent is rationed to one focal element per view. `--accent-btn` is the AA-safe solid-button fill. rgb tuples `--accent-rgb` / `--marble-rgb` feed the canvases.

**Type:** **Geist** = `--font-sans` (all UI + reading/body; `--font-serif` is re-pointed to Geist). **IBM Plex Mono** = `--font-mono-display` / `--font-display` (brand, titles, labels, `.label-mono`, the wordmark). Fraunces + Spectral are removed. Fonts load in `app/layout.tsx`.

**The dot-matrix is the ONLY decorative language — the constellation/night-sky idea is killed.** Everything decorative is dots.

**The dynamism DNA (`lib/dots.ts`) — applies to EVERY dot element, always:**
- Motion is carried by **SIZE + OPACITY + COLOUR**, NOT by moving dots (position may shift only slightly).
- Always-on and **CHAOTIC** (`chaos(seed,t)` — per-dot random freq/phase, not coherent waves). Never static.
- The cursor is a **local lens**: it brightens + enlarges nearby dots with per-dot entropy (no halo); large + gentle on desktop; **transient on touch** (deactivate on pointerup/cancel for non-mouse).
- Dots are reasonably big; lattice density is responsive (denser on mobile). A faint-dot cull keeps it fast.

**The elements (all share the DNA):**
- **Logo = Socrates' FACE** as a dot-matrix (cropped to the face, oval-clipped, lonely dots pruned). Generated by `scripts/gen-bust.mjs` → `components/brand/bust-dots.ts` (`BUST_DOTS_COARSE` for the small SVG mark/favicon `app/icon.svg`/`app/apple-icon.png`; `BUST_FACE` density grid for the landing field). Small mark: `components/brand/bust-mark.tsx` (SVG + shimmer).
- **Wordmark** `components/brand/wordmark.tsx`: bust mark + `SOCRATES AI` in Plex caps, with "AI" + a blinking block cursor in the blue accent.
- **Landing centerpiece** `components/decor/landing-field.tsx` (`LandingField`): ONE unified full-viewport canvas — ambient dots + a concentric gradient swelling toward the centerpiece + the FACE **emerging from the same lattice** (sampled from `BUST_FACE`). Face ⇄ **orb** morph is PURE size/opacity/colour (no dot moves); the orb uses the "speaking" rhythm (concentric rings), is enlarged, and appears at random for a minority of the time (face-dominant). Hero is left (face) / right (wordmark + 2-line tagline + buttons), one viewport tall, with a diffused scrim behind the text.
- **Ambient background** `components/decor/dot-matrix.tsx` (`DotMatrix`): faint field on `/login` and the authenticated app shell.
- **Orb (Socrates' presence in voice)** `components/sky/breathing-star.tsx` (`BreathingStar`, canvas; states `idle|listening|speaking|thinking|ended`, each visibly distinct/dynamic).
- **Bank "field"** `components/sky/constellation.tsx`: entries as dots, NO connecting lines.
- **Live reference:** the `/design` route documents the whole system.

**Messaging:** lead with the big idea — *an instrument for the examined life that sharpens your OWN thinking* (AI aiding human cognition; a midwife for wisdom). Voice/chat are entry points, not the headline. Anti-metric ethos (no streaks/counts/gamification) is law.
